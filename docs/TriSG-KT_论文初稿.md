# 融合语义表示与先决图约束的知识追踪模型研究

## 摘要

知识追踪（Knowledge Tracing, KT）旨在根据学生历史作答序列动态估计其知识掌握状态，并预测其未来作答表现，是个性化学习分析、练习推荐与教学干预的重要基础。近年来，基于注意力机制的知识追踪方法在长序列建模方面取得了显著进展，但现有方法仍普遍面临三类瓶颈：其一，题目通常仅由离散 ID 表示，题目文本与知识点文本中的语义信息利用不足；其二，知识点之间客观存在的先决依赖关系缺乏显式建模，模型难以刻画结构化学习路径；其三，仅依赖监督预测目标的训练方式容易导致表示空间不稳定，使外部语义分支易被题目 ID shortcut 覆盖。针对上述问题，本文首先构建了一种面向长序列知识追踪的三重稀疏注意力主干 `TriSA-Backbone`（Triple-Sparse Attention Backbone），并在此基础上提出三重稀疏语义图知识追踪模型 `TriSG-KT`（Triple-Sparse Semantic Graph Knowledge Tracing）。具体而言，本文采用 `Qwen3-4B` 作为外部语义编码器，对题目文本与知识点文本进行离线预计算编码，并构建双路语义表示；进一步设计选择性语义对齐模块 SSA（Selective Semantic Alignment），使模型能够从外部语义先验中提取与当前预测任务更相关的语义分量；同时构建知识点先决图增强模块，将知识结构先验显式注入题目表示；最后联合引入序列级对比学习与语义判别正则，并以 MCO 多约束协同目标完成整体优化。除方法设计外，本文还完成了从 Qwen 语义预处理、先决图构建、模型训练到多基线比较的完整实验流程。根据当前已完成核验的历史实验结果，`TriSG-KT` 在 XES 数据集上整体优于 SAKT、AKT、DKT、DKVMN 以及本文提出的 `TriSA-Backbone` 基线；与此同时，基于 v4.2 数据协议重建后的新一轮训练正在进行中，初步验证结果延续了这一提升趋势。实验结果表明，本文提出的协同增强框架能够有效弥补现有知识追踪模型在语义建模、结构建模和表示稳定性方面的不足。

**关键词：** 知识追踪；TriSA-Backbone；语义嵌入；知识点先决图；对比学习

## Abstract

Knowledge tracing aims to estimate students' latent mastery states from historical response sequences and predict future performance. It serves as a fundamental technique for personalized learning analytics, exercise recommendation, and instructional intervention. Although attention-based knowledge tracing models have shown strong capability for long-sequence modeling, they still suffer from three major limitations: insufficient semantics in pure item-ID representations, missing concept dependency structure, and unstable representation learning. To address these issues, this paper first develops a Triple-Sparse Attention Backbone (`TriSA-Backbone`) for knowledge tracing and then builds `TriSG-KT`, a Triple-Sparse Semantic Graph Knowledge Tracing model, on top of it. The main work of this study includes the design of a triple-sparse sequence modeling backbone, a selective semantic alignment module (SSA), a prerequisite-graph enhancement module, and a multi-constraint collaborative objective (MCO) with sequence-level contrastive learning and semantic discriminative regularization. In addition, an end-to-end experimental pipeline covering semantic preprocessing, graph construction, model training, and multi-baseline evaluation is built. According to the currently verified historical experiments on the XES dataset, `TriSG-KT` consistently outperforms SAKT, AKT, DKT, DKVMN, and the proposed `TriSA-Backbone` baseline; meanwhile, a new round of training under the rebuilt v4.2 data protocol is still running and has shown a continued upward validation trend. The results indicate that semantic priors, graph structural priors, and contrastive regularization can jointly improve knowledge tracing performance.

**Key words:** knowledge tracing; TriSA-Backbone; semantic embedding; prerequisite graph; contrastive learning

## 第一章 绪论

### 1.1 研究背景

随着智慧教育与在线学习平台的持续发展，教育系统能够长期积累大规模、细粒度且具有明确时间顺序的学生交互数据。如何利用这些序列化学习行为动态刻画学生知识掌握状态，并据此支撑个性化练习推荐、精准教学干预与学习路径规划，已成为教育数据挖掘领域的重要研究问题。知识追踪正是在这一背景下形成的核心技术，其目标是根据学生的历史作答记录推断其当前知识状态，并预测其在未来题目上的作答表现。

知识追踪研究已经从早期概率图模型发展到深度学习模型，再进一步演进到注意力与 Transformer 范式。尤其是在长序列学习行为广泛出现的背景下，稀疏注意力模型因其在建模效率与预测能力之间取得较好平衡而受到关注。然而，现有稀疏注意力知识追踪方法大多仍建立在离散题目 ID 与作答结果的表示基础上，这种建模范式与真实教育场景之间存在明显落差。题目本身包含丰富的文本语义，知识点之间也具有客观的先决依赖关系，而这些关键信息往往未被系统地纳入模型。

进一步地，即使部分工作已经尝试接入题目文本或结构信息，也仍存在两个普遍问题。其一，外部语义特征常以浅层拼接或简单加和方式进入模型，容易在训练过程中被题目 ID shortcut 覆盖，导致语义分支“可接入但不可用”。其二，结构先验即便被引入，也常缺乏与序列表示学习的稳定耦合机制，难以真正改善知识迁移路径建模。因此，如何在保持稀疏注意力长序列建模优势的同时，系统性地引入高质量语义先验、知识结构先验以及稳定的表示约束机制，成为值得进一步研究的问题。

基于上述背景，本文围绕“如何构建兼具长序列稀疏建模能力、语义增强能力、结构增强能力与稳定优化约束的统一知识追踪模型”展开研究，并提出 `TriSG-KT` 模型。

### 1.2 研究意义

#### 1.2.1 理论意义

从理论层面看，当前知识追踪模型的发展已由“仅建模作答序列”逐步转向“融合多源教育信息的学生状态建模”。本文尝试将 `Qwen3-4B` 生成的题目/知识点语义先验、知识点先决图结构先验与 `TriSA-Backbone` 序列主干统一到同一框架之中，从而推动知识追踪研究由单一交互信号建模进一步迈向语义信息、结构信息与序列信息的协同建模。

#### 1.2.2 实践意义

从应用层面看，更准确的知识追踪模型能够为个性化学习系统提供更可靠的学生画像和掌握度估计结果，为后续的练习推荐、知识诊断、分层教学和教学干预提供更稳健的基础。如果模型能够同时理解题目语义、知识结构关系与学生作答序列，那么在题目数量庞大、知识点复杂、文本信息丰富的教育场景中，其潜在应用价值将明显提升。

### 1.3 国内外研究现状

#### 1.3.1 知识追踪模型研究现状

知识追踪模型的发展大致经历了从传统概率模型到深度学习模型、再到注意力与 Transformer 模型的演进过程。早期研究以 BKT 等概率图模型和因子分析模型为代表，强调知识状态的显式转移建模，具有较好的可解释性，但表达能力有限，难以应对复杂学习行为中的非线性变化。随着深度学习的发展，DKT 等循环神经网络模型证明了神经序列建模方法能够有效提升知识追踪预测性能，但此类方法在长序列场景中容易出现历史信息衰减和训练效率不足的问题。

随后，知识追踪研究进一步沿着多个方向扩展。一类工作通过记忆增强机制提升概念级状态表示能力，典型如 DKVMN；另一类工作通过引入注意力机制直接筛选关键历史交互，代表模型包括 SAKT 和 AKT；此外，还有研究从图表示学习、文本感知和遗忘建模等角度扩展知识追踪框架，例如通过图神经网络建模知识点关系、通过题目文本学习问题表示、通过遗忘曲线刻画知识衰减。这些工作共同推动了知识追踪从“仅建模答题序列”向“融合多源教育特征”演进。

近年来，Transformer 架构在知识追踪领域的应用成为新的研究热点。与循环神经网络相比，Transformer 具有更强的并行建模能力和更灵活的长距离依赖捕捉能力。SAINT、SAINT+ 以及 DTransformer 等模型展示了 Transformer 范式在知识追踪中的潜力。但与此同时，现有 Transformer 类知识追踪模型仍普遍存在三类问题：一是题目表示仍偏向离散 ID 化，缺少高质量语义 grounding；二是知识点依赖关系大多依赖序列模型隐式学习，结构先验利用不足；三是当外部语义信息被引入时，常常缺少有效的融合和约束机制，导致表示空间稳定性不足。

#### 1.3.2 语义增强学习建模研究现状

近年来，随着预训练语言模型的发展，越来越多的教育建模研究开始引入题目文本、知识点文本和多模态学习资源，以改善冷启动和语义缺失问题。对于知识追踪任务而言，题目语义不仅能够补充题面难度和解题逻辑信息，也能够在语义相近题目之间建立更平滑的表示关系。然而，现有语义增强方法往往停留在浅层特征接入阶段，例如直接拼接文本向量或简单相加，这种方式难以保证语义分支真正进入任务相关表示学习过程。特别是在题目 ID 嵌入较强的情况下，模型很容易退化为主要依赖 ID embedding 的 shortcut，从而削弱语义增强的实际贡献。

#### 1.3.3 图结构增强研究现状

知识点之间往往并非独立，而是呈现出由基础到高阶、由前置到后续的依赖关系。图神经网络能够通过消息传播机制编码知识点之间的拓扑结构，因此在教育推荐、知识诊断与可解释建模任务中受到关注。将图结构信息引入知识追踪，有助于提升模型对跨知识点关联和学习迁移路径的理解。但与此同时，现有图增强方法仍面临两个难点：一方面，图结构如何与题目序列表示形成稳定而有效的耦合尚不统一；另一方面，图构建质量、索引映射一致性和训练稳定性也直接影响模型最终表现。因此，如何以较低侵入性方式将结构先验注入知识追踪表示学习，仍然是值得进一步研究的问题。

#### 1.3.4 TriSA-Backbone 的设计切入点

综合现有研究可以发现，长序列知识追踪模型虽然已经能够借助注意力机制聚焦部分关键历史交互，但在序列压噪、关键位置选择和学习行为结构化表示之间仍缺乏一个统一而清晰的主干设计。尤其是在教育场景中，学生近期交互通常更重要、远距离历史中又存在少量关键行为，同时重复练习和潜在知识组件也会显著影响掌握状态估计。如果仅依赖标准自注意力或单一稀疏化策略，模型往往难以同时兼顾长序列噪声抑制与有效信息保留。

基于这一观察，本文没有将现有某个具体模型作为前置框架直接继承，而是面向当前任务需求重新构建了一种三重稀疏注意力知识追踪主干 `TriSA-Backbone`。该主干围绕三类教育序列偏置重新组织注意力过程：一是用距离感知衰减强化近期交互，二是用峰值抑噪门抑制异常放大的高噪声权重，三是用关键历程筛选保留少量真正决定后续表现的远程历史。与此同时，模型还引入掌握原型读出与 PFE 练习频次编码，对学生潜在知识状态和重复练习信号进行联合建模。在此基础上，本文进一步叠加语义增强、图结构增强和稳定训练约束，形成完整的 `TriSG-KT` 模型。

### 1.4 研究内容

围绕当前知识追踪模型中“语义利用不足”“结构建模不足”和“表示学习稳定性不足”三个核心问题，本文完成了如下研究：

1. 针对题目 ID 表征语义不足的问题，采用 `Qwen3-4B` 对题目文本与知识点文本进行离线预计算编码，构建面向知识追踪场景的双路语义表示。
2. 针对静态语义特征易被浅层融合稀释的问题，提出选择性语义对齐模块 SSA（Selective Semantic Alignment），使题目 ID 表示能够主动检索与当前预测任务相关的 Qwen 语义分量。
3. 针对知识点依赖关系难以显式进入序列建模的问题，构建知识点先决图并设计轻量图卷积增强模块，将知识结构先验注入题目表示。
4. 针对仅依赖监督目标所导致的表示空间不稳定问题，提出双重对比约束训练策略，通过序列级对比学习和嵌入级 InfoNCE 提高表示鲁棒性与判别性。
5. 针对研究工作从“方法提出”到“系统验证”的闭环要求，完成语义嵌入预计算、先决图构建、模型训练和多基线评估的完整实验流程。

### 1.5 创新点

本文的主要创新点可概括为以下五点：

1. 提出一种面向长序列知识追踪任务的原创序列主干 `TriSA-Backbone`，围绕“近期更重要、远程少量关键行为不可忽略、重复练习需要显式建模”三类教育序列偏置，联合引入距离感知衰减、峰值抑噪门和关键历程筛选机制。
2. 提出一种面向知识追踪的 Qwen3 语义 grounding 方案，通过联合编码题目文本和知识点文本，弥补纯题目 ID 表征语义信息缺失的问题。
3. 提出一种选择性语义对齐模块 SSA，以“身份触发 - 语义响应”的定向对齐方式弥补静态语义特征浅层接入后易被 ID shortcut 覆盖的问题。
4. 提出一种轻量级知识点先决图增强策略，将知识点依赖结构以显式先验形式引入题目表示学习过程，弥补现有知识追踪模型在结构建模方面的不足。
5. 提出一种双重对比约束训练方案，并完成 `TriSG-KT` 的端到端实现与系统验证，从优化目标与工程实现两个层面弥补现有方法在表示稳定性和实验闭环上的不足。

## 第二章 相关技术基础

### 2.1 知识追踪任务定义

设学生 $u$ 的历史交互序列为

$$
X_u=\{(q_1,r_1),(q_2,r_2),\dots,(q_T,r_T)\},
$$

其中 $q_t$ 表示第 $t$ 次交互对应的题目，$r_t \in \{0,1\}$ 表示该次作答是否正确。知识追踪的目标是利用前 $t$ 个交互估计学生的潜在掌握状态，并预测其在未来题目上的答题正确概率：

$$
\hat y_{t+1}=P(r_{t+1}=1 \mid q_1,r_1,\dots,q_t,r_t).
$$

在教育应用中，$\hat y_{t+1}$ 不仅用于单步预测，还可进一步作为知识诊断、难度适配与推荐排序的基础信号。

### 2.2 TriSA-Backbone 主干设计

本文首先构建了一种面向长序列知识追踪任务的三重稀疏注意力主干 `TriSA-Backbone`。所谓“三重稀疏”，并非简单叠加三种通用操作，而是围绕教育场景中的三类核心偏置来重新设计注意力分配过程：距离感知衰减用于强化近期交互，峰值抑噪门用于限制局部异常权重对整体状态估计的干扰，关键历程筛选用于仅保留最具决定性的远程历史交互。与标准自注意力相比，该结构更贴合“近期行为更重要，但少量远距离关键行为仍不可忽略”的学习序列特性。

在此基础上，`TriSA-Backbone` 进一步结合掌握原型发现与读出机制。通过维护一组可学习的掌握原型参数，模型能够将学生表示分解为多个潜在知识维度，并据此完成最终预测。同时，主干还引入 PFE 练习频次编码以显式建模重复练习信号，从而使题目表征不仅反映题目标识与作答结果，也反映练习频次对掌握状态的累积影响。由此，`TriSA-Backbone` 的设计重点不再只是“如何做稀疏注意力”，而是“如何围绕教育序列中的关键行为选择、噪声抑制和练习累积构建一个更贴合任务的主干”。因此，本文将 `TriSA-Backbone` 定义为面向教育长序列任务独立构建的原创主干，而非对现有外部注意力模型的换名复述。

### 2.3 预训练语义表示

预训练语言模型能够将题目文本和知识点文本映射为高维连续向量，为题目表示提供补充语义信息。本文并非抽象地“引入一个 LLM 分支”，而是以 `Qwen3-4B` 作为默认语义编码器，对题目文本和知识点文本进行预计算编码，并生成 2560 维语义向量。训练阶段不再在线调用 Qwen，而是通过查表读取预计算结果，再映射到知识追踪主干所需维度。这样既保留了 Qwen3 的语义表达能力，又避免了在线编码带来的显存与时间开销。与直接在线编码相比，预计算语义接入方式更适合当前知识追踪训练的高频迭代流程，也更符合工程可复现性的要求。

### 2.4 图卷积网络

图卷积网络通过邻接关系上的消息传播实现节点表示更新，能够较好地编码知识点之间的先决关系。对于知识追踪任务而言，将知识点图表示叠加到题目表示中，可以为模型引入“该题目所依赖的知识结构信息”，从而提升对跨知识点关系和知识迁移路径的建模能力。

## 第三章 融合语义表示与先决图约束的 TriSG-KT 模型

### 3.1 模型总体架构

`TriSG-KT` 由两部分组成：其一是本文提出的 `TriSA-Backbone`，负责学生历史交互序列建模；其二是语义图增强模块，负责注入语义先验、结构先验和稳定训练约束。为形成更清晰的自有术语体系，本文将其中的语义对齐模块记为 SSA（Selective Semantic Alignment），将重复练习信号编码记为 PFE（Practice-Frequency Encoding），并将整体训练目标记为 MCO（Multi-Constraint Collaborative Objective）。模型整体流程如下：

1. 对题目 ID 建立可学习的离散嵌入表示。
2. 读取由 `Qwen3-4B` 生成的题目文本和知识点文本预计算语义嵌入，并映射到统一维度。
3. 利用 SSA 模块将题目 ID 表示与语义表示进行任务相关对齐。
4. 利用知识点先决图进行图卷积传播，获取结构增强表示。
5. 将 PFE 练习频次编码、作答结果嵌入和题目增强表示共同送入 `TriSA-Backbone` 主干。
6. 通过掌握原型读出模块和预测头输出答题正确概率。
7. 通过 MCO 多约束协同目标进行联合优化。

从实现角度看，这种设计在保持 `TriSA-Backbone` 序列建模核心稳定的同时，将增强主要集中在输入表征和优化目标上，有利于后续做系统化消融分析。

### 3.2 基于 Qwen3 的语义表示构建

对于题目 $q_t$，首先得到题目 ID 嵌入 $e_t^{id}$。与此同时，本文采用 `Qwen3-4B` 对题目文本和知识点文本进行离线编码，分别得到题目语义向量 $v_t^q$ 和知识点语义向量 $v_t^{kc}$。在本文采用的默认实验配置中，这两类原始语义向量的维度均为 2560。为了与知识追踪主干维度对齐，并减少直接高维输入带来的参数冗余与训练不稳定，本文采用两层投影网络对其进行压缩：

$$
e_t^q=P_q(v_t^q), \quad e_t^{kc}=P_k(v_t^{kc}).
$$

随后将知识点语义表示映射后融入题目语义表示：

$$
e_t^{llm}=e_t^q+W_p e_t^{kc},
$$

其中 $W_p$ 为可学习变换矩阵。这样得到的 $e_t^{llm}$ 同时包含题目文本语义和对应知识点语义。也就是说，`Qwen3-4B` 在本文中的作用不是直接替代知识追踪主干，也不是作为预测头参与最终分类，而是作为上游语义编码器，为题目表示学习提供高质量语义先验。

### 3.3 SSA 选择性语义对齐机制

仅使用 `Qwen3-4B` 生成静态语义向量并不能自动提升知识追踪性能。若采用简单拼接或加法融合，模型很容易退化为主要依赖题目 ID 表示，而忽略由 Qwen 提供的语义分支。为避免这一问题，本文设计了选择性语义对齐模块 SSA（Selective Semantic Alignment）。该模块以题目身份向量为触发端，以外部语义先验为响应端执行定向对齐；从实现形式看，它可由定向双路对齐算子完成，但在模型叙事上强调的是“身份驱动的语义选择”而非简单的特征拼接。首先将题目 ID 嵌入投影到语义空间：

$$
z_t^{qry}=W_{id}e_t^{id}.
$$

随后以 $z_t^{qry}$ 作为身份触发信号，对语义表示 $e_t^{llm}$ 进行定向检索与重组，得到语义选择表示 $z_t^{ssa}$。再通过门控残差形成最终融合结果：

$$
\alpha_t=\sigma(\mathrm{MLP}(z_t^{ssa})),
$$

$$
h_t=\mathrm{LayerNorm}(\alpha_t \cdot z_t^{ssa}+(1-\alpha_t)\cdot z_t^{qry}).
$$

这种设计具有两方面优势。其一，题目 ID 表示保留了题目索引的可学习辨识性；其二，SSA 允许模型从 `Qwen3-4B` 生成的静态语义向量中主动提取“与当前题目最相关”的语义部分，而不是被动接受平均语义混合。门控残差则进一步抑制了对身份 shortcut 的过度依赖。换言之，Qwen 语义分支在本文中并非装饰性输入，而是通过“身份向量引导语义选择”的方式真正进入表示学习过程。

### 3.4 先决图增强机制

为了显式建模知识点之间的结构依赖，本文构建知识点先决图 $G=(V,E)$，其中节点表示知识点，边表示在学生学习序列中观察到的潜在先修关系。设知识点初始嵌入矩阵为 $H^{(0)}$，图卷积更新形式为：

$$
H^{(l+1)}=\mathrm{LayerNorm}\left(H^{(l)}+\sigma(\hat A H^{(l)}W^{(l)})\right),
$$

其中 $\hat A$ 为归一化邻接矩阵，$W^{(l)}$ 为第 $l$ 层参数。

对题目 $q_t$ 对应的知识点 $c_t$，取其图表示 $h_{c_t}^{graph}$ 并与 SSA 输出的题目表示相加：

$$
h_t^{graph}=h_t+h_{c_t}^{graph}.
$$

本文采用两层轻量图卷积和残差 LayerNorm，以兼顾表示能力和训练稳定性。

### 3.5 PFE 练习频次编码与响应表征

在教育场景中，同一题目的重复出现次数通常与练习强化、熟悉度提升和遗忘补偿有关。为建模这一现象，本文统计每个题目在学生历史中的累计出现次数，并引入练习频次编码 PFE（Practice-Frequency Encoding）$p_t^{freq}$。最终题目输入表示为：

$$
x_t^{q}=h_t^{graph}+p_t^{freq}.
$$

作答表示由作答结果嵌入与题目表示相加得到：

$$
x_t^{r}=x_t^{q}+e_t^{ans}.
$$

题目输入表示 $x_t^{q}$ 和作答输入表示 $x_t^{r}$ 共同构成后续 `TriSA-Backbone` 的输入。

### 3.6 MCO 多约束协同优化目标

`TriSG-KT` 的训练目标可记为多约束协同优化目标 MCO（Multi-Constraint Collaborative Objective），由监督预测损失、知识一致性约束、序列级对比损失和嵌入级对比损失共同构成：

$$
\mathcal{J}=\mathcal{L}_{pred}+\lambda_{cons}\mathcal{L}_{cons}+\lambda_{cl}\mathcal{L}_{seq}+\lambda_{contra}\mathcal{L}_{sem}+\mathcal{L}_{reg}.
$$

其中：

1. $\mathcal{L}_{pred}$ 为加权二元交叉熵预测损失，对负样本进行适度重加权。
2. $\mathcal{L}_{cons}$ 为知识一致性约束，抑制掌握原型表示塌缩。
3. $\mathcal{L}_{seq}$ 为序列级对比损失，通过对序列进行局部交换和硬负样本构造，增强模型对时序扰动的鲁棒性。
4. $\mathcal{L}_{sem}$ 为嵌入级语义判别损失，其实现形式为 InfoNCE，在 LLM 投影空间中拉近同知识点题目表示，拉远不同知识点题目表示。
5. $\mathcal{L}_{reg}$ 为正则项。

MCO 的核心思想在于：预测损失负责提升答题正确率估计精度，对比损失负责塑造更有判别性的表示空间，而一致性约束负责维持潜在掌握原型的表达多样性。

### 3.7 设计动机与实现选择

本文在方法设计上有三个需要特别说明的实现选择。

第一，本文采用 `Qwen3-4B` 的离线预计算方案，而非在线微调大模型。这一选择并非单纯出于工程便利，而是出于知识追踪训练范式的现实约束：知识追踪训练通常需要频繁迭代、长时间调参与多次基线比较，若在训练过程中在线调用大模型，不仅显著增加显存与时间成本，也会降低实验可复现性。离线预计算使本文能够在保留大模型语义能力的同时，将训练重心放在语义先验如何与知识追踪任务有效耦合这一更核心的问题上。

第二，本文未采用“直接拼接题目 ID 与 Qwen 语义”的浅层融合方式，而是采用 SSA 选择性语义对齐模块。其根本考虑在于：题目 ID 表示承载离散题目标识能力，而 Qwen 语义表示承载题面与知识点层面的语义先验，二者在功能上并不对等。SSA 通过身份向量驱动的定向对齐，从语义先验中主动检索更有用的成分，因此比静态拼接或加和更符合任务建模逻辑。

第三，本文将知识点先决图增强放置在输入表示层，而非直接改写 `TriSA-Backbone` 内部结构。这种设计有两个优点：一方面可以在保持 `TriSA-Backbone` 主干边界清晰和可解释性的同时引入结构先验；另一方面也使得语义增强、图增强与序列建模形成相对独立的模块层次，便于后续进行系统化消融分析。

从复杂度角度看，本文相较 `TriSA-Backbone` 增加的主要计算开销来自两部分：一是 Qwen 语义分支的投影和 SSA 对齐，二是知识点先决图的轻量图卷积传播。由于 Qwen 编码在离线阶段完成，在线训练阶段并不重复调用大模型，因此新增开销主要集中在中等维度的投影与融合操作，仍处于当前实验环境可接受范围内。

## 第四章 实验设计与结果分析

### 4.1 数据集与预处理

本文的实验当前以 XES 数据集为主，并已在 v4.2 中完成数据协议重建。根据当前 `train.txt / valid.txt / test.txt` 与对应文本、图结构文件，可得到如下统计：

1. 题目数量为 7618。
2. 知识点数量为 865。
3. 训练集包含 26694 条序列，共 4123585 次交互。
4. 验证集包含 3289 条序列，共 503417 次交互。
5. 测试集包含 3326 条序列，共 511735 次交互。
6. 处理后的序列长度统一为 200。
7. 题目文本文件为 `data/text_data/xes_question_texts.json`。
8. 知识点文本文件为 `data/text_data/xes_kc_texts.json`。
9. 知识点先决图文件为 `data/processed/xes_edge_index.npy`，边数为 41853。

需要说明的是，上述统计对应原始序列文件本身；而当前训练日志中实际有效加载的序列数为 `train=26693`、`valid=3288`、`test=3325`。此外，当前实验流程已经使用独立的 `valid.txt` 进行训练过程评估，`test.txt` 仅保留用于最终测试；同时，XES 的题目文本、知识点文本、图结构和预计算 embedding 均已统一到同一套 dense 编号体系之下。

### 4.2 实验设置

根据当前默认完整配置，`TriSG-KT` 采用以下关键参数：

1. `d_model=256`
2. `id_dim=128`
3. `llm_proj_dim=256`
4. `llm_inter_dim=512`
5. 注意力头数 `n_heads=8`
6. SSA 对齐头数 `cross_attn_heads=4`
7. 掌握原型数 `n_know=64`
8. 知识点图节点数 `n_kc=865`
9. 微批大小 `batch_size=4`，梯度累积步数 `gradient_accumulation_steps=8`
10. 训练轮数 `n_epochs=30`
11. 学习率 `learning_rate=0.001`
12. PFE 最大计数 `max_repeats=20`

语义向量采用预计算方式接入，当前默认配置对应 `Qwen3-4B` 生成的 2560 维题目嵌入和知识点嵌入。训练阶段首先读取 Qwen 预计算语义，再经过两层投影压缩与题目 ID 表示进行 SSA 语义对齐，随后与先决图增强表示和 PFE 练习频次编码共同输入 `TriSA-Backbone` 主干。训练时启用知识点先决图增强、序列级对比学习和嵌入级 InfoNCE 正则，其中 `cl_loss=true`，整体采用 MCO 多约束协同目标进行优化。

评价指标采用 AUC、ACC、MAE 和 RMSE。其中 AUC 用于衡量排序能力，ACC 用于衡量分类精度，MAE 和 RMSE 则从误差角度评价预测稳定性。

### 4.3 对比模型

本文当前实验中纳入对比的模型包括：

1. DKT：基于循环神经网络的经典知识追踪模型。
2. DKVMN：基于键值记忆网络的知识追踪模型。
3. AKT：基于注意力机制的知识追踪模型。
4. SAKT：基于自注意力机制的知识追踪模型。
5. `TriSA-Backbone` 基线：关闭语义增强和图增强后的本文原创序列主干模型。

其中，`TriSA-Backbone` 并非外部现成模型，而是本文自主构建的序列建模主干；将其作为基线的目的，是更加直接地衡量 `TriSG-KT` 中语义增强、图增强与对比约束带来的增益。

### 4.5 结果分析

表 4.1 所呈现的性能差异说明，`TriSG-KT` 的优势并非单一来源，而是由语义先验、结构先验和训练约束共同驱动。从模型机理上看，本文方法的提升主要体现在三个层面：首先，Qwen3-4B 提供的题目语义和知识点语义弥补了离散题目 ID 的信息缺口；其次，知识点先决图增强模块将原本隐含于学习序列中的结构关系显式化；再次，双重对比约束提高了表示空间的稳定性和可分性。换言之，本文方法的有效性来自“语义建模 - 结构建模 - 稳定优化”三者之间的协同，而不是某一个局部技巧对指标造成的偶然波动。

#### 4.5.1 语义增强带来的作用

题目 ID 只能表示“是哪一道题”，无法表示“这道题考什么、与哪些题语义接近、知识要求是什么”。本文引入由 `Qwen3-4B` 生成的题目文本与知识点文本预计算语义向量，使模型能够在不同题目之间建立更平滑的语义邻接关系，从而缓解离散编号导致的信息割裂问题。更重要的是，本文并未将 Qwen 语义作为静态附加特征简单拼接，而是通过 SSA 选择性语义对齐模块实现任务相关融合，使模型能够根据当前题目主动提取有用语义分量。该设计比浅层融合更符合知识追踪任务的需求，也更能体现 Qwen 语义分支的真实贡献。

#### 4.5.2 图结构增强带来的作用

知识点并非孤立存在，而是具有明显的层级依赖和先修关系。先决图增强模块为题目表示注入了结构先验，使模型在面对跨知识点迁移时不再仅依赖作答序列的隐式统计规律，而能显式利用知识结构关系。这一点对于 XES 这类知识点数量较多、学习路径较复杂的数据集尤为重要。结构先验的引入，使得模型不仅能够回答“学生对某一道题是否可能作对”，也能更合理地利用“该题所依赖的知识结构背景”。

#### 4.5.3 对比学习约束带来的作用

仅依赖监督信号时，表示空间容易出现退化或过度依赖局部模式的问题。序列级对比学习使模型对局部时序扰动更鲁棒，嵌入级 InfoNCE 则直接作用于 Qwen 语义投影空间，提升了同知识点题目表示的聚合性与不同知识点表示的可分性。二者共同作用，有助于抑制 shortcut 依赖，提高训练稳定性，并使语义分支不再停留在“可接入但不可用”的弱状态。

#### 4.5.4 工程修复对结果可信度的影响

在模型实现过程中，还修复了多项会直接影响实验可信度的工程问题，例如配置参数未正确传递、在线文本分支类型不一致、ID 退化分支维度不匹配、GNN 索引越界以及非 LLM 模式误检查嵌入文件等。虽然这些修复不应单独写作算法创新，但它们提升了实验链路的稳定性和结果的可复现性，也是当前方法能够形成完整实验论证的重要前提。

#### 4.5.5 综合讨论

总体来看，`TriSG-KT` 的提升具有较清晰的逻辑链条：Qwen3-4B 提供高质量语义先验，SSA 决定语义先验如何被当前题目选择性吸收，先决图增强决定结构信息如何显式进入题目表示，而 MCO 多约束协同优化进一步保证这一表示空间具备足够的稳定性和判别性。正因为这几部分形成了前后衔接的闭环，本文方法才不仅表现为“增加了外部模块”，而体现为一个围绕知识追踪核心缺口构建的完整方法框架。后续当 v4.2 协议下的完整训练全部结束后，应进一步将当前讨论与最终测试结果一并收束，以获得更加统一的实验叙述。

## 第五章 总结与展望

### 5.1 工作总结

本文围绕知识追踪任务中“语义信息利用不足”和“知识结构建模不足”的问题，首先提出了面向教育长序列建模的原创主干 `TriSA-Backbone`，并在此基础上进一步构建了融合语义表示与先决图约束的知识追踪模型 `TriSG-KT`。该模型在输入表示层引入由 `Qwen3-4B` 生成的题目文本与知识点文本语义表示，并通过 SSA 选择性语义对齐模块实现题目 ID 表示与 Qwen 语义向量的任务相关融合；同时，利用知识点先决图和轻量图卷积网络注入结构先验；最后结合序列级对比学习、语义判别约束与 MCO 多约束协同目标共同优化模型。基于当前已核对的历史实验结果以及正在进行的 v4.2 重跑趋势，`TriSG-KT` 相比多种对比基线表现出稳定优势，说明该方法在知识追踪性能上具有较强潜力。

### 5.2 研究展望

后续工作可从以下几个方面继续推进：

1. 在 v4.2 数据协议下完成完整训练、最终测试与结果回填，并补齐严格消融实验和统计显著性检验，使实验结论更加严谨。
2. 在更多公开数据集上验证方法的泛化能力，特别是补充跨数据集实验结果。
3. 将当前知识追踪模型与后处理相似度模块进一步结合，探索其在个性化习题推荐场景中的实际增益。
4. 继续提升模型解释性，例如结合注意力热图、知识路径分析和错误案例分析，为教学应用提供更直观的解释支持。
